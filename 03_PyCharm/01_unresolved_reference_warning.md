# solution for unresolved reference warning on PyCharm
```
root directory
-> mouse right click
-> Mark Directory as 
-> Source Root

Preferences
-> Build, Execution, Deployment
-> Console
-> Python Console
-> Configure Interpreters
-> Add source roots to PYTHONPATH
```
